Python Asyncio Http (aiohttp） Demo
==============
用于开发简单的web 后端服务， 单核并发可达7000-8000

Installation
============

Clone repo and install library::

    $ git clone git@github.com:aio-libs/aiohttp-demos.git
    $ cd aiohttp-demos

Install the app::

    $ cd aiohttp-demo
    $ pipenv install
    # pipenv run python manager.py
